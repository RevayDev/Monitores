# Guía de Aprendizaje para el Proyecto

Para dominar este proyecto y el desarrollo web moderno, te recomiendo estudiar los siguientes temas:

## 1. JavaScript Moderno (ES6+)
- **ES Modules**: Entender `import` vs `require`.
- **Async/Await**: Cómo manejar promesas y peticiones asíncronas.
- **Destructuring & Spread Operator**: Limpiar y copiar objetos/arrays.

## 2. React.js
- **Hooks**: `useState` para datos y `useEffect` para efectos secundarios (como cargar datos al iniciar).
- **Context API**: Cómo compartimos datos entre componentes (ej. `ToastContext`).
- **React Router**: Manejo de navegación entre páginas.

## 3. Node.js & Express
- **Middleware**: Entender cómo `cors` y `body-parser` procesan la información.
- **Layered Architecture**: Por qué separamos Controllers de Services.

## 4. Bases de Datos SQL (MySQL)
- **Consultas Básicas**: `SELECT`, `INSERT`, `UPDATE`, `DELETE`.
- **Relaciones**: Cómo conectar tablas mediante IDs.
- **Connection Pooling**: Por qué no abrimos una conexión nueva por cada usuario.

## 5. Herramientas
- **Vite & NPM**: Gestión de dependencias y servidores de desarrollo.
- **Git**: Control de versiones para no perder tu trabajo.
